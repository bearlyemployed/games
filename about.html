Bearly
Employed.

Bearly Employed is a one-person iOS game studio. Every pixel, every note, every line of code is made solo - no team, no investors, no ads.

The goal is simple: build small, polished, ad-free mobile games that I'd actually want to play. Games that respect your time and your intelligence. Games that are just... fun.

The stack

SwiftUI
iOS
Game Center
StoreKit 2
Python
Xcode

The philosophy

No ads. Ever. I believe ads in mobile games are a fundamentally broken experience and I won't build that. If a game needs to be sustained, it'll be through optional purchases that feel fair - cosmetics, convenience, never pay-to-win.

Offline first. Every game works without an internet connection because your time shouldn't depend on signal strength.

Built for all ages. Games should be accessible and welcoming. No dark patterns, no countdown timers designed to create anxiety, no gotcha mechanics.
